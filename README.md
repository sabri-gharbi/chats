# Create T3 App

This is a [T3 Stack](https://create.t3.gg/) project bootstrapped with `create-t3-app`.

## What's next? How do I run this on my local machine ?

-   copy ".env.example" to ".env"
-   npm i
-   npm run dev

## How do I deploy this?

Follow guides for [Vercel](https://create.t3.gg/en/deployment/vercel),
[Netlify](https://create.t3.gg/en/deployment/netlify) and [Docker](https://create.t3.gg/en/deployment/docker) for more
information.
